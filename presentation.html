Technical Deep Dive

The Agentic Loop

  • 1
    Memory — previous score + rationale injected into system prompt so the LLM reasons about what changed, not just the current snapshot
  • 2
    Concurrent tool fetch — yfinance (price/fundamentals), Finnhub (news sentiment), SEC EDGAR (filings) all fetched in parallel via asyncio.gather — no waiting
  • 3
    Structured output — LLM must emit exactly AI_RISK_SCORE / AI_RECOMMENDATION / RATIONALE. Regex parser with 3 fallback strategies extracts values even from noisy output
  • 4
    Self-reflection — LLM critiques its own answer. If score delta > threshold it revises. Stops early when it converges
  • 5
    Guardrails — 12 regex patterns detect leaked prompt templates. <think> blocks stripped. Score-derived fallback if rationale is garbage
